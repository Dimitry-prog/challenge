# Проект: Куда я поеду

Данный проект - практическое задание Яндекс Практикума на соревновании по верстке.
В проекте используется следующее:

1. HTML семантика;
2. БЭМ методология;
3. Кроссбраузерность;
4. Подходы к позиционированию: flex, position и grid;
5. Адаптивность;
6. JavaScript.
